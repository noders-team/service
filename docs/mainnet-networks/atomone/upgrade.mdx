## Clone Atomone repository

```js
cd $HOME
rm -rf atomone
git clone https://github.com/atomone-hub/atomone.git
cd atomone
git checkout v2.0.0
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.atomone/cosmovisor/upgrades/v2.0.0/bin
mv build/atomoned ~/.atomone/cosmovisor/upgrades/v2.0.0/bin/
rm -rf build
```

# Without Cosmovisor

## Clone Atomone repository

```js
cd $HOME
rm -rf atomone
git clone https://github.com/atomone-hub/atomone.git
cd atomone
git checkout v2.0.0
```

## Build binaries

```js
make install
```

## Restart node and check logs

```js
sudo systemctl restart atomoned && sudo journalctl -u atomoned -f --no-hostname -o cat
```
