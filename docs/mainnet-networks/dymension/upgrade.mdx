## Clone Dymension repository

```js
cd $HOME
rm -rf dymensionxyz
git clone https://github.com/dymensionxyz.git
cd dymensionxyz
git checkout v3.2.0
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.dymension/cosmovisor/upgrades/v3.2.0/bin
mv build/dymd ~/.dymension/cosmovisor/upgrades/v3.2.0/bin/
rm -rf build
```

# Without Cosmovisor

## Clone Dymension repository

```js
cd $HOME
rm -rf dymensionxyz
git clone https://github.com/dymensionxyz.git
cd dymensionxyz
git checkout v3.2.0
```

## Build binaries

```js
make install
```

## Restart node and check logs

```js
sudo systemctl restart dymd && sudo journalctl -u dymd -f --no-hostname -o cat
```
