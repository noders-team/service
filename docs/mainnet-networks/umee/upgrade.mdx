## Clone UX repository

```js
cd $HOME
rm -rf umee-network
git clone https://github.com/umee-network.git
cd umee-network
git checkout auto
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.umee/cosmovisor/upgrades/auto/bin
mv build/umeed ~/.umee/cosmovisor/upgrades/auto/bin/
rm -rf build
```

# Without Cosmovisor

## Clone UX repository

```js
cd $HOME
rm -rf umee-network
git clone https://github.com/umee-network.git
cd umee-network
git checkout auto
```

## Build binaries

```js
make install
```

## Restart node and check logs

```js
sudo systemctl restart umeed && sudo journalctl -u umeed -f --no-hostname -o cat
```
