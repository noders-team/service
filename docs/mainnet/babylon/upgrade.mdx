## Clone Babylon repository

```js
cd $HOME
rm -rf babylon
git clone https://github.com/babylonlabs-io/babylon.git
cd babylon
git checkout v2.2.0
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.babylond/cosmovisor/upgrades/v2.2.0/bin
mv build/babylond ~/.babylond/cosmovisor/upgrades/v2.2.0/bin/
rm -rf build
```

# Without Cosmovisor

## Clone Babylon repository

```js
cd $HOME
rm -rf babylon
git clone https://github.com/babylonlabs-io/babylon.git
cd babylon
git checkout v2.2.0
```

## Build binaries

```js
make install
```

## Restart node and check logs

```js
sudo systemctl restart babylond && sudo journalctl -u babylond -f --no-hostname -o cat
```
