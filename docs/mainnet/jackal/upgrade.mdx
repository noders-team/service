## Clone Jackal repository

```js
cd $HOME
rm -rf canine-chain
git clone https://github.com/JackalLabs/canine-chain.git
cd canine-chain
git checkout v4.6.0
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.canine/cosmovisor/upgrades/v4.6.0/bin
mv build/canined ~/.canine/cosmovisor/upgrades/v4.6.0/bin/
rm -rf build
```

# Without Cosmovisor

## Clone Jackal repository

```js
cd $HOME
rm -rf canine-chain
git clone https://github.com/JackalLabs/canine-chain.git
cd canine-chain
git checkout v4.6.0
```

## Build binaries

```js
make install
```

## Restart node and check logs

```js
sudo systemctl restart canined && sudo journalctl -u canined -f --no-hostname -o cat
```
