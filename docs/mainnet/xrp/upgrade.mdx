## Clone XRP repository

```js
cd $HOME
rm -rf node
git clone https://github.com/xrplevm/node.git
cd node
git checkout v8.0.0
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.exrpd/cosmovisor/upgrades/v8.0.0/bin
mv build/exrpd ~/.exrpd/cosmovisor/upgrades/v8.0.0/bin/
rm -rf build
```

# Without Cosmovisor

## Clone XRP repository

```js
cd $HOME
rm -rf node
git clone https://github.com/xrplevm/node.git
cd node
git checkout v8.0.0
```

## Build binaries

```js
make install
```

## Restart node and check logs

```js
sudo systemctl restart exrpd && sudo journalctl -u exrpd -f --no-hostname -o cat
```
