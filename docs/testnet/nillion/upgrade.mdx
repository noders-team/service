## Clone Nillion repository

```js
cd $HOME
rm -rf nilchain
git clone https://github.com/NillionNetwork/nilchain.git
cd nilchain
git checkout v0.2.5
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.nillionapp/cosmovisor/upgrades/v0.2.5/bin
mv build/nilchaind ~/.nillionapp/cosmovisor/upgrades/v0.2.5/bin/
rm -rf build
```

# Without Cosmovisor

## Clone Nillion repository

```js
cd $HOME
rm -rf nilchain
git clone https://github.com/NillionNetwork/nilchain.git
cd nilchain
git checkout v0.2.5
```

## Build binaries

```js
make install
```

## Restart node and check logs

```js
sudo systemctl restart nilchaind && sudo journalctl -u nilchaind -f --no-hostname -o cat
```
