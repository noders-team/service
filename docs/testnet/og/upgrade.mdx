## Clone ZeroGravity repository

```js
cd $HOME
rm -rf 0g-chain
git clone https://github.com/0glabs/0g-chain.git
cd 0g-chain
git checkout v0.2.0-alpha.4-752-g764df4768
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.0gchain/cosmovisor/upgrades/v0.2.0-alpha.4-752-g764df4768/bin
mv build/0gchaind ~/.0gchain/cosmovisor/upgrades/v0.2.0-alpha.4-752-g764df4768/bin/
rm -rf build
```

# Without Cosmovisor

## Clone ZeroGravity repository

## Restart node and check logs

```js
sudo systemctl restart 0gchaind && sudo journalctl -u 0gchaind -f --no-hostname -o cat
```
