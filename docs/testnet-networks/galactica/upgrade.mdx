## Clone Galactica repository

```js
cd $HOME
rm -rf networks
git clone https://github.com/Galactica-corp/networks.git
cd networks
git checkout v0.2.7
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.galactica/cosmovisor/upgrades/v0.2.7/bin
mv build/galacticad ~/.galactica/cosmovisor/upgrades/v0.2.7/bin/
rm -rf build
```

# Without Cosmovisor

## Clone Galactica repository

```js
cd $HOME
rm -rf networks
git clone https://github.com/Galactica-corp/networks.git
cd networks
git checkout v0.2.7
```

## Build binaries

```js
make install
```

## Restart node and check logs

```js
sudo systemctl restart galacticad && sudo journalctl -u galacticad -f --no-hostname -o cat
```
