## Clone XRP repository

```js
cd $HOME
rm -rf node
git clone https://github.com/xrplevm/node.git
cd node
git checkout 8.0.0-rc.1
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.exrpd/cosmovisor/upgrades/8.0.0-rc.1/bin
mv build/exrpd ~/.exrpd/cosmovisor/upgrades/8.0.0-rc.1/bin/
rm -rf build
```

# Without Cosmovisor

## Clone XRP repository

```js
cd $HOME
rm -rf node
git clone https://github.com/xrplevm/node.git
cd node
git checkout 8.0.0-rc.1
```

## Build binaries

```js
make install
```

## Restart node and check logs

```js
sudo systemctl restart exrpd && sudo journalctl -u exrpd -f --no-hostname -o cat
```
