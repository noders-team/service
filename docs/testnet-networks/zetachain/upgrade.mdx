## Clone Zetachain repository

```js
cd $HOME
rm -rf zeta-chain
git clone https://github.com/zeta-chain.git
cd zeta-chain
git checkout v30.0.0
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.zetacored/cosmovisor/upgrades/v30.0.0/bin
mv build/zetacored ~/.zetacored/cosmovisor/upgrades/v30.0.0/bin/
rm -rf build
```

# Without Cosmovisor

## Clone Zetachain repository

```js
cd $HOME
rm -rf zeta-chain
git clone https://github.com/zeta-chain.git
cd zeta-chain
git checkout v30.0.0
```

## Build binaries

```js
make install
```

## Restart node and check logs

```js
sudo systemctl restart zetacored && sudo journalctl -u zetacored -f --no-hostname -o cat
```
