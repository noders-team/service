## Clone Side repository

```js
cd $HOME
rm -rf sideprotocol
git clone https://github.com/sideprotocol.git
cd sideprotocol
git checkout v2.0.0-rc.6
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.side/cosmovisor/upgrades/v2.0.0-rc.6/bin
mv build/sided ~/.side/cosmovisor/upgrades/v2.0.0-rc.6/bin/
rm -rf build
```

# Without Cosmovisor

## Clone Side repository

```js
cd $HOME
rm -rf sideprotocol
git clone https://github.com/sideprotocol.git
cd sideprotocol
git checkout v2.0.0-rc.6
```

## Build binaries

```js
make install
```

## Restart node and check logs

```js
sudo systemctl restart sided && sudo journalctl -u sided -f --no-hostname -o cat
```
