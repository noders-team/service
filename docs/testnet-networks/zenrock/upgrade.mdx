## Clone Zenrock repository

```js
cd $HOME
rm -rf zrchain
git clone https://github.com/Zenrock-Foundation/zrchain.git
cd zrchain
git checkout v6.13.0
```

## Build binaries

```js
make build
```

## Prepare binaries for Cosmovisor

```js
mkdir -p ~/.zrchain/cosmovisor/upgrades/v6.13.0/bin
mv build/zenrockd ~/.zrchain/cosmovisor/upgrades/v6.13.0/bin/
rm -rf build
```

# Without Cosmovisor

## Clone Zenrock repository

```js
cd $HOME
rm -rf zrchain
git clone https://github.com/Zenrock-Foundation/zrchain.git
cd zrchain
git checkout v6.13.0
```

## Build binaries

```js
make install
```

## Restart node and check logs

```js
sudo systemctl restart zenrockd && sudo journalctl -u zenrockd -f --no-hostname -o cat
```
